# Dialogue System

Um sistema de diálogos prático, modular e de fácil integração, totalmente compatível e otimizado para a **Unity 6 (6000.4.0f1+)**.

## 🚀 Como Instalar (Unity Package Manager)

Você pode instalar este sistema diretamente no seu projeto Unity usando o Git URL:

1. Abra seu projeto na **Unity 6**.
2. No menu superior, vá em **Window > Package Manager**.
3. Clique no botão **`+`** (canto superior esquerdo) e selecione **Add package from git URL...**.
4. Cole o link abaixo e clique em **Add**:
   ```text
   [https://github.com/JMNumber3/](https://github.com/JMNumber3/)[Dialogue-System-for-Unity-6000.4-Dev-U-].git
